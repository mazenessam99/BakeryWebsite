## SEO & Accessibility

* Include meta `description` for each page. 
* Add Open Graph and meta tags for rich sharing.
* Use semantic HTML (`<main>`, `<header>`, `<nav>`, `<article>`).
* Use `alt` text for all images and `loading="lazy"` for non-critical images.
